# Finance_Tracker
As tech stack changes will update

A modern personal finance platform designed to help users gain complete visibility into their financial health. The application provides tools for expense tracking, budgeting, investment portfolio management, financial goal planning, and advanced analytics through an intuitive dashboard.
The project is being built as a real-world SaaS-style application to demonstrate modern software engineering practices, including clean architecture, authentication and authorisation, data visualisation, background processing, testing, and cloud deployment.
Core Features Financial Management Income and expense tracking Transaction categorization Recurring transactions Budget creation and monitoring Financial goal tracking Investment Tracking Portfolio management Asset allocation monitoring Investment performance tracking Net worth calculations Analytics & Reporting Spending trends and insights Budget performance dashboards Cash flow analysis Monthly and yearly financial reports Future Enhancements CSV bank statement imports OCR receipt scanning AI-powered transaction categorisation AI financial assistant Automated reporting and notifications

Backend ASP.NET Core Entity Framework Core SQL Server
